Ce plugin permet aux visiteurs de votre blog de commenter très simplement de manière anonyme, s'ils le désirent.

Attention, même en désactivant les commentaires anonymes, il est évidemment toujours possible pour un visiteur d'utiliser un pseudo et une fausse adresse e-mail.

Autoriser les commentaires anonymes
Permet d'activer ou de désactiver globalement les commentaires anonymes.
Nom de remplacement
Le nom qui sera utilisé pour les commentaires anonymes.
E-mail de remplacement
L'adresse e-mail qui sera utilisée pour les commentaires anonymes.
